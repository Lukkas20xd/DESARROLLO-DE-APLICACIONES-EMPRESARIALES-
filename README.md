# Registro de Clientes

Aplicación simple para registrar, almacenar y listar clientes.

Campos: Nombre, RUT/ID, Correo, Teléfono.

Tecnología:
- Interfaz: Streamlit (Python)
- Almacenamiento: SQLite (archivo local `clientes.db`)

Cómo ejecutar:


1. Instalar dependencias:

```powershell
pip install -r requirements.txt
```

2. Ejecutar la app:

```powershell
py -m streamlit run app.py
```

Archivos principales:
- `app.py` — código fuente de la aplicación.
- `requirements.txt` — dependencias.
- `clientes.db` — creado automáticamente al ejecutar la app.
